FLEX

Ширину блока flex можно задать в % с таким кодом

					.text {
						display: flex;
						align-items: center;
						flex: 0 0 494/1240 * 100%;
					}

Позволяем объекту flex увеличиваться. Если поставить на контейнер, то внутри элементы будут на всю ширину контейнера

flex: 1 1 auto;
width: 100%; - блоку внутри контейнера, внутри которого еще flex элементы

Свойство: flex: 1; = flex:1 1 0n; (где n - единица длины).

flex-grow: Число, определяющее, насколько элемент будет расти относительно остальных элементов.
flex-shrink Число, указывающее, насколько элемент будет уменьшаться относительно остальных гибких элементов.
flex-basis Длина элемента. Правовые значения: "авто", "наследовать" или число, за которым следуют "%", "px", "em" или любой другой блок длины.
